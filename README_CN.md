# Bilibili 低质视频过滤器 (Bilibili Low-Effort Video Filter)

这是一个基于 Tampermonkey 的油猴脚本，旨在帮助你识别并标记 Bilibili 上的低质量搬运视频、AI 生成的“幻灯片”视频或营销号内容。

本脚本**不会**直接删除视频，而是将视频封面替换为**浅红色警告块**。这样你既能一眼识别出低质内容，又保留了点击观看的选择权，防止误删你想看的内容。

## 🚀 核心功能
- **颜色家族分析**：检测封面中白色、黑色或灰色占比是否超过 80%（这是低质搬运视频的典型特征）。
- **中心加权逻辑**：自动忽略封面边缘 15% 的区域，有效过滤那些企图通过“彩色边框”或“毛玻璃边框”绕过检测的视频。
- **视觉警告**：将视频封面变红，并在标题前添加 `[⚠️]` 标签。
- **不破坏布局**：保持页面整洁，被标记的视频卡片依然可以正常点击跳转。

## 🛠️ 安装方法
1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展。
2. [点击此处安装脚本](https://raw.githubusercontent.com/YOUR_GITHUB_USERNAME/YOUR_REPO_NAME/main/bilibili-low-effort-filter.user.js)。
3. 在弹出的 Tampermonkey 页面中点击“安装”或“更新”。

## ⚙️ 工作原理
脚本利用 Canvas 直方图分析技术对封面进行“扫描”。它会通过预设的偏移量跳过封面四周的边框，仅计算中心区域的单色像素比例。

你可以通过修改脚本顶部的 `CONFIG` 对象来调整过滤强度：
- `FAMILY_THRESHOLD`: 触发过滤所需的单色比例（默认 0.80，即 80%）。
- `BORDER_INSET`: 忽略边框的宽度比例（默认 0.15，即忽略四周各 15% 的面积）。

## 🐞 调试与反馈
如果脚本误标或漏标了某些视频，你可以按 `F12` 打开浏览器开发者控制台。脚本会记录每一个被标记视频的标题、视频链接和封面原图链接，方便你根据这些数据调整 `CONFIG` 参数。
